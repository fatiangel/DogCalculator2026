# DogCalculator2026
# 妙麗(柯基犬)歲數計算機 🐶

一個使用 **HTML + CSS + JavaScript** 製作的狗狗年齡計算器。  
輸入狗狗出生日期後，可以計算：

- 狗的實際年齡
- 換算後的人類年齡

本專案同時示範：

- 日期計算
- JavaScript 數學運算
- LocalStorage 資料儲存
- 基本前端 UI 設計

---

# 📖 專案介紹

輸入狗狗出生日期後，系統會自動計算：

- 🐕 狗的實際年齡
- 👤 對應的人類年齡
- 📚 年齡換算公式說明

並提供以下功能：

- **開始計算**
- **清除資料**

此外，輸入的出生日期會儲存在 **LocalStorage** 中，重新整理頁面後仍會保留。
